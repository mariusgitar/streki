# Illustrasjonsmaker

Illustrasjonsmaker er en intern kommunal webapp for å beskrive og generere illustrasjoner fra tekst.

## Stack

- React
- Vite
- Tailwind CSS
- Netlify Functions
- OpenRouter API

## Struktur

- `src/components/` for React-komponenter
- `src/utils/` for hjelpefunksjoner
- `netlify/functions/` for serverless-funksjoner

Ingen API-funksjonalitet er implementert ennå.
